## Task 2: AI Tool-Building Challenge
**File:** `task-2-ai-tool/index.html` — Open in any browser. Zero installation needed.

### What it does
A Customer 360 Insight Tool that aggregates data from 4 sources (CRM, Support Tickets, Emails, Slack) into a unified view with:
- Account summary with risk badge (Healthy / Needs Attention / At Risk)
- CRM details (deal stage, value, owner)
- Support tickets (priority-coded)
- Email history (sentiment-coded)
- Automated signal & risk detection
- Prioritized next best action recommendations

### How to use
1. Open `task-2-ai-tool/index.html` in Chrome/Firefox/Safari/Edge
2. Select a customer from the dropdown
3. View the 360° dashboard

### Documentation
Full details in `task-2-ai-tool/documentation.md` including:
- Problem statement
- Dummy data description
- Logic/workflow diagram
- Example input/output
- 5 improvement ideas
